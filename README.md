# A2UI Inspector

Cross-renderer browser-based debugger for A2UI v0.9 message streams.

```bash
npx a2ui-inspector
```

See `docs/superpowers/specs/2026-05-21-a2ui-inspector-design.md` for the design.

## Development

```bash
pnpm install
pnpm test
pnpm dev
```

## Status

Phase 1 MVP — in development.
